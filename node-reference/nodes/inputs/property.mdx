---
title: "Property"
description: "Select a property from a JSON object"
---

<img src="/images/propertyInput.png" style={{ borderRadius: '0.5rem' }} />

The `property` input allows you to select a single property from an `object`. 
You can select specific child values, or parent objects and arrays. 
If you provide a list of input objects, you will need to specify the same key for all objects. 

<Info>
If you need more control over object property selection, use the `Get Key` tool from the Objects library.
</Info>
