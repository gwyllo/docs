**Image Preview**

Connect an image url to download and preview the image. Useful for text-image runchats. 

**Load GLB**

Connect a glb or gltf url to download and preview the model. Useful for image-3D runchats.

**Upload Screenshot**

Uploads a screenshot of the selected Rhino screenshot to runchat and returns the url for use with generate modeels

**Camera Projection**

Project an image onto a mesh using the camera in the active Rhino viewport
