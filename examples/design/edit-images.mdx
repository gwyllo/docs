### Inputs

`concept`

Describe how you want to edit the image.

### Outputs

`result`

The edited image

## Use and applications

The prompt node will accept multiple prompts to operate on a single input image.
This can be used to explore different ideas for editing the image in parallel.
You can also generate variations on your prompts using the prompt node and an array output,
see [the image variations example](/examples/design/create-image-variations) for reference.
